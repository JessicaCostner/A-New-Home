Knoxville's Elders

This web page will be designed to gather information from all of the animal shelters in Knoxville. It will serve as a linking page to the shelters. It will gather the images, names, ages, and species of the available pets in the Knoxville area. Purpose: The purpose of this web page is to gather all of the information from the local animal shelters in one place. For those who are looking to adopt a new pet for their family, they can simply go to our webpage and view all of the available animals in the Knoxville area. This will save them time and effort by gathering all of the animals in one location, instead of looking through multiple shelter pages. Technical Specifications: This will be usable on computers only. We will not be making this project usable on mobile devices. It will be hosted through the shell client on the Pellissippi server. Content: We will be getting information from the local animal shelters in Knoxville. We will be using Java code to run a Web Crawler. It will be used to search the web pages for different species, along with their name and age. Style: The style of the page will be modified using HTML, CSS, and the other languages we will be learning in class. Function: We will be storing data within a database so that the load time between pages will be shortened. The database will be updated regularly so that the content of the page is accurate. Requirements: This application will gather information about adoptable pets from local animal shelters in the area. They will be sorted by species and the user will be able to select a tab at the top of the page that will allow them to view the available dogs, cats, ect. After selecting a species, it will take them to a new page that will list the animals within that species with their name, age, picture and a link to the shelter they are at.
